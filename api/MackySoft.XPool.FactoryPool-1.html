    Int32 capacity

    The pool capacity. If less than or equal to 0, ArgumentOutOfRangeException will be thrown.

    Func<T> onCreate

    Method that create new instance. If is null, ArgumentNullException will be thrown. This method is must return not null. If returns null, Rent() throw NullReferenceException.

    Action<T> onRent

    Callback that is called when Rent() is successful.

    Callback that is called when Return(T) is successful.

    Callback that is called when capacity is exceeded and the instance cannot be returned to the pool.
